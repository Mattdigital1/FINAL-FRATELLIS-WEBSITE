TIPS
----
- JPG/JPEG works fine. PNG also OK. WebP best for performance.
- Recommended size: 1200–1600 px on the long edge. The site will crop
  to the right aspect ratio automatically (4:3 for dish cards, 4:5 for
  split-media). No need to crop in advance.
- File size: aim for < 300 KB each. Use https://squoosh.app or similar
  to compress without visible quality loss.
- If you want to swap any photo later, just replace the file with the
  same filename and refresh the page.
